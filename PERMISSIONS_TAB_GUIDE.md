# Cloud Run 権限タブの見つけ方

## 手順

### 方法1: サービス詳細ページから

1. **Cloud Run サービスページを開く**
   ```
   https://console.cloud.google.com/run?project=tamron-cloudrun-prod-new
   ```

2. **サービス一覧から `ai-drbfm-backend` をクリック**

3. **サービス詳細ページが開きます**
   - 上部にタブが表示されます：
     - **概要** (Overview)
     - **ログ** (Logs)
     - **YAML** (YAML)
     - **権限** (Permissions) ← これです

4. **「権限」タブをクリック**

5. **「プリンシパルを追加」ボタンをクリック**

### 方法2: 直接URLから

以下のURLに直接アクセスしてください：

```
https://console.cloud.google.com/run/detail/asia-northeast1/ai-drbfm-backend/permissions?project=tamron-cloudrun-prod-new
```

### 方法3: 左側のメニューから

1. サービス詳細ページを開く
2. 左側のメニュー（ナビゲーションメニュー）に「権限」という項目があります
3. それをクリック

## 権限設定の手順

1. **「プリンシパルを追加」ボタンをクリック**

2. **以下の情報を入力**
   - **新しいプリンシパル**: `allUsers` と入力
   - **ロール**: ドロップダウンから `Cloud Run 起動元` を選択
     - または `roles/run.invoker` と入力

3. **「保存」をクリック**

## 注意事項

- 「権限」タブが見つからない場合は、ページをリロードしてみてください
- ブラウザのキャッシュをクリアする必要がある場合もあります
- プロジェクトの権限によっては「権限」タブが表示されない場合があります

## スクリーンショットの説明

権限タブは通常、サービス詳細ページの上部に以下のようなタブとして表示されます：

```
[概要] [ログ] [YAML] [権限] [メトリクス] [トレース]
```

「権限」タブをクリックすると、現在のIAMポリシーが表示され、「プリンシパルを追加」ボタンが表示されます。
